### A SIMPLE TODO APP.

---

- Clone repository

  ```
  #clone this repository
  git clone git@git.dongnguyen.dev:dongnguyenltqb/todo.git
  
  #jump to folder todo
  cd todo
  
  #get dependency
  go mod download
  ```

- Install

  ```
  #install to $GOBIN
  go install
  ```

  

- Help

  ```
  #get help
  todo -h
  
  #get list help command
  todo list -h
  
  #get add help command
  todo add -h
  ```
